Conservator's Note

First launch on macOS

Why the first open gets blocked

Free & open source, but not signed with a paid Apple cert — so macOS blocks the very first open. Any one of these clears it for good:

  • 1Right-click the app → Open → Open.
  • 2Or System Settings → Privacy & Security → Open Anyway.
  • 3Or clear the quarantine flag from a terminal:
zsh — clear quarantine
$ xattr -dr com.apple.quarantine "/Applications/Glyph Grid Studio.app"
